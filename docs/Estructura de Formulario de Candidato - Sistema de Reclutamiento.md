# **Estructura Sugerida para Formulario de Reclutamiento**

Este documento detalla los campos necesarios para capturar la información de un candidato de manera estructurada. Está diseñado para que la recolección de datos sea fluida para el usuario y fácilmente traducible a una estructura de base de datos técnica.

## **1\. Información Personal y de Contacto**

Estos campos identifican al individuo y permiten establecer comunicación.

- **Nombre(s):** Nombres de pila del candidato.
- **Apellidos:** Apellidos completos.
- **Correo Electrónico:** Dirección de e-mail personal o profesional.
- **Teléfono de Contacto:** Número telefónico con código de país/área.
- **Perfil de LinkedIn:** Enlace directo a su red profesional.
- **Portafolio o Sitio Web:** Enlace a muestras de trabajo (especialmente para roles creativos o técnicos).

## **2\. Información Demográfica y Ubicación**

Datos útiles para logística y cumplimiento legal.

- **Fecha de Nacimiento:** Utilizada para verificar la edad legal y estadísticas.
- **Ciudad de Residencia:** Ubicación actual.
- **País:** País donde reside actualmente.
-

## **3\. Perfil Profesional y Experiencia**

Esta sección es el núcleo del formulario para evaluar la aptitud del candidato.

| Campo                     | Descripción en Lenguaje Natural                                                    |
| :------------------------ | :--------------------------------------------------------------------------------- |
| **Título Profesional**    | Cómo se define el candidato profesionalmente (ej. "Ingeniero de Software Senior"). |
| **Empresa Actual/Última** | Nombre de la organización más reciente donde trabajó.                              |
| **Años de Experiencia**   | Cantidad total de años de experiencia en el sector relevante.                      |
| **Resumen de Perfil**     | Un breve párrafo donde el candidato describe su valor y experiencia principal.     |

##

## **4\. Educación y Habilidades**

- **Nivel Máximo de Estudios:** Grado académico alcanzado (Bachillerato, Licenciatura, Maestría, PhD).
- **Campo de Estudio:** Carrera o área de especialización.
- **Idiomas:** Lista de idiomas y nivel de dominio (A1, A2, B1, B2, C1, C2).
- **Habilidades Técnicas (Hard Skills):** Herramientas, software o lenguajes de programación específicos.

## **5\. Disponibilidad y Expectativas**

Datos críticos para el cierre de la contratación.

- **Salario actual bruto / neto**
- **Pretensión Salarial:** Salario bruto mensual o anual esperado (especificar moneda).
- **¿Cómo se enteró de la vacante?:** Fuente del reclutamiento (Referido, LinkedIn, Portal de empleo, etc.).

## **6\. Documentación Adjunta**

- **Currículum Vitae (CV):** Archivo principal (usualmente PDF).
